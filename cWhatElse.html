What Else You Need

You don't need any other programs to use Tunnelblick. It comes with the OpenVPN program and all required drivers built into the Tunnelblick application itself.

However, you do need three or four other things:

  • Administrator access to your computer so the program and configurations can be made secure and OpenVPN can make network configuration changes needed to establish a VPN.

  • Access to a VPN server, because your computer is only one end of the VPN tunnel — the VPN server is the other end.

  • Configuration information. You would get this from whomever/whatever provides you with access to a VPN server.

  • (Optional) You may also need a passphrase and/or a username and password to access the VPN. You would get them from whomever/whatever provides you with access to a VPN server.

For more information about getting access to a VPN server, see Getting VPN Service.

You get configuration information from who/whatever provides you with access to a VPN server. It is either

  • A Tunnelblick VPN Configuration (Tunnelblick 3.1 and up), or

  • An OpenVPN configuration (.ovpn or .conf file), plus any files with keys, or certificates, or scripts that the configuration file references.
