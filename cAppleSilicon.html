Tunnelblick and Apple Silicon

The Tunnelblick application itself should work on Apple Silicon systems using Rosetta 2.

However:

  • Tun configurations on Apple Silicon systems will work only if they don't require Tunnelblick's Tun system extension. See The Future of Tun and Tap VPNs on macOS for instructions on how to make sure a configuration does not require Tunnelblick's Tun system extension and to modify the OpenVPN configuration file so that it does not require it.

  • Tap configurations will not work on Apple Silicon systems and there are no plans for Tunnelblick to support such setups.

Both Tun and Tap configurations work on Intel systems on macOS up to and including the initial release of macOS macOS 11 "Big Sur". However, Apple may remove the ability to use Tap configurations at any time. See The Future of Tun and Tap VPNs on macOS for details.

At some point the developers anticipate releasing a "Universal" version of Tunnelblick which runs natively on Intel and Apple Silicon but that may not be done until the anticipated release of a the next major version of macOS in 2021. It will not include the ability to use Tap configurations on Apple Silicon systems.
