Deprecated Downloads

Deprecated means "superseded and should be avoided".

Please don't use a deprecated version of Tunnelblick unless you cannot use the latest stable or beta version.

If you are having problems with the latest stable version of Tunnelblick, please try the latest beta version.

Please report problems on the Tunnelblick Discussion Group.

Binary releases of other versions are not publicly available. Anyone requiring such a release may build it from source using the instructions included in the source code or may contact the developers for a copy.

In 2018 there were false reports of malware in Tunnelblick. See No Malware In Tunnelblick for details.

For macOS Tiger (10.4) through Mavericks (10.9)

This is a security and stability update for an old version of Tunnelblick. It should be used only on macOS 10.4 - 10.9.
On macOS 10.13 and higher, the latest stable or beta version from the Downloads page should be used.

Notes:

  • You may not be able to download Tunnelblick binaries using Safari on old versions of macOS, even though they should run on macOS 10.4 - 10.9. You may be able to download using Firefox or another browser, or you can download it on a newer system and then install it on on older system.

  • The digital signature is valid only on macOS 10.8.5 and earlier.

These versions contain serious security vulnerabilities.

If you have trouble installing 3.5.26, try installing 3.5.25 first, and then installing 3.5.26 (by double-clicking the Tunnelblick icon in the disk image, as usual):
