Contributing to the Tunnelblick Project

Thank you for considering making a contribution to support Tunnelblick.

We are no longer accepting donations of money.

Please consider helping Tunnelblick in other ways:

  • Our biggest need is for people to test and report problems with Tunnelblick or suggest improvements.

  • We also need help translating Tunnelblick, especially for languages which are not yet completely translated (see Translation Status).

  • If you are an experienced Tunnelblick or OpenVPN user or a networking expert, you can help by answering questions in the Tunnelblick Discussion Group.

  • If you have skills in programming or image creation and manipulation, you can help us develop Tunnelblick - email developers@tunnelblick.net for more information.
